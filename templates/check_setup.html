To configure Google OAuth:

  1. Go to Google Cloud Console
  2. Create or select your project
  3. Configure the OAuth consent screen:
    • Set User Type to External
    • Add required app information
    • Add scopes: .../auth/userinfo.email, .../auth/userinfo.profile, .../auth/calendar.events
  4. Create OAuth 2.0 Client ID credentials
  5. Add the redirect URI shown above to authorized redirect URIs
  6. Enable the Google Calendar API
  7. Set your CLIENT_ID and CLIENT_SECRET as environment variables
